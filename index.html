About Me

This is the portfolio website for Mathilde Mouw-Rao. My artwork spans music, code, and the visual arts. As a singer-songwriter, I write and perform under the name Tilde Mouw. I have also written and produced an opera, Love and a Lion (2008), and a musical, No Catharsis (2013).

I work as a full-time software engineer writing code in Ruby, Javascript, HTML and CSS, and using the Rails and Angular frameworks.
